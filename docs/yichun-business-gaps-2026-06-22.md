## 0. 核心结论

**V1.0 是"骨架完整、血肉缺失"的最小可用版本**——能上线、能跑通流程、能审核、能有用户——但**没有 IM 接不住用户，没有 SEO 接不住流量，没有商业化接不住团队**。

### 三个致命断层
1. **成交闭环断** — 手机号脱敏 + 无 IM，用户登录后联系不到卖家，转化率 < 5%
2. **流量入口裸奔** — sitemap / TDK / URL / 主动推送全 0，百度搜不到 = 等于没存在
3. **商业化 = 0** — 公告系统是唯一变现路径；伊春年广告盘子仅 100-300 万，没付费 = 团队养不活

> **伊春不缺一个分类信息平台，缺的是 0458.cn 的"现代化 + 微信化 + 商业化"版本**。
> 抓住林下经济 / 旅游季节 / 林业职工这三个伊春独有场景，就是和 58 / 百姓网最大的差异化护城河。

### 1.2 角色覆盖盲区

PRD US-001~007 已覆盖求职者、房东、二手卖家、招聘方、审核员、运营、管理员。**还缺**：
- **中介**（房产中介、招聘中介） — 没有商家主页支撑
- **买房者**（二手房 vs 二手物品混在一起） — 体验差
- **政府/旅游/林场**信息发布方 — 缺 B 端入口
- **老用户**（45-65 岁林业退休职工） — UI 大字模式、流程极简化

### 1.3 业务流程缺失

- **没有 IM** → 唯一转化通道 = 电话 → 但电话号又脱敏了 → **死循环**
- **没有商家入口** → 招聘方以个人身份发 → 没有信用沉淀 → 第二天被举报下架
- **没有实名认证** → 黑产/小号成本极低 → 出现黑中介/虚假房源/招嫖 → 平台社死

### 2.3 SEO 缺失清单

| 项 | 影响 | 修复成本 |
|---|---|---|
| 无 sitemap.xml | 百度不知道你有啥页面 | 1 天 |
| 无 25 分类 TDK | 长尾词全部浪费 | 1 周 |
| 无 12 区县 TDK | "伊春 XX 区 XX" 长尾全部丢 | 0.5 周 |

### 2.4 微信生态缺失

- **公众号菜单**：0 → 1 天开发 → 30%+ 复访流量
- **H5 跳转 + 海报生成**：2 天 → 每个帖子自带传播机制
- **分享带参（utm_source / 分享者 ID）**：1 天 → 后续可做分销/积分
- **微信小程序**：V2 → 门槛高、审核严，先用公众号 H5 跑通闭环

## 3. 搜索

### 3.1 当前 FULLTEXT + ngram 的问题

| 问题 | 影响 | 修复成本 |
|---|---|---|
| 林场名称分词不准（"乌伊岭"被拆成"乌""伊""岭"）| 搜索召回率 -30% | 2 天（自定义词典） |
| 无自动补全（用户输入 2 字就开始搜）| 用户找不到热门词 | 1 天（Redis 前缀树）|
| 无纠错（同音字、形近字）| 用户搜错就空白 | 2 天（编辑距离）|
| 无筛选器面板 | 招聘 / 二手 / 租房共用一套筛选 = 没筛选 | 3 天（按 type 切换）|
| 无相关推荐（搜完空白页）| 用户跳出 | 1 天 |
| 无地图搜索 | 找房效率低 | 1.5 天（高德 SDK）|
| 无搜索历史 | 用户重复输入 | 0.5 天 |
| 无热门搜索榜单 | 新用户引导 | 0.5 天 |

**搜索 V2 全套升级 = 2 周**，是平台从"能搜"到"搜得好"的关键跳跃。

### 3.2 招聘 / 二手 / 租房的差异化搜索

| 模块 | 关键筛选维度 | 当前支持 |
|---|---|---|
| 招聘 | 行业 + 薪资 + 学历 + 经验 | ⚠️ 部分 |
| 二手 | 价格区间 + 成色 + 自提范围 | ❌ 无 |
| 租房 | 户型 + 租金 + 小区 + 配套设施 | ⚠️ 部分 |
| 便民 | 区域 + 分类 | ✅ |

**共用一套筛选 = 三边都不讨好**。必须按 type 切换筛选器。

## 4. 商业化 / 赚钱模型

### 4.1 战略纠偏

> **PRD 把商业化推到 V2 是最大战略错误**。
> "等日活 5000 再做商业化"是鸡和蛋——**没付费就别想有日活，没日活就别想有付费**。

### 4.2 商业化路径时间表

| 时间 | 变现手段 | 预期月营收 |
|---|---|---|
| **+1 月** | Banner 广告（首页 + 4 个分类列表页）| 几百块 |
| **+3 月** | 付费置顶（招聘 + 租房）| 几千块 |
| **+6 月** | VIP 会员（招聘方 + 商家）| 1-2 万 |
| **+12 月** | 商家入驻费 + 担保交易抽佣 + 微信小程序流量分成 | 5 万+ |

### 4.3 最小可行商业化（V1.0.5 / Phase 2 前置）

```
Day 1-2:  Banner 位（首页 + 4 个分类列表页，100 元/周、50 元/周）
Day 3-5:  付费置顶（招聘 / 租房 / 便民，9.9 元/天，列表前 3 位）
Day 6-8:  商家入驻（免费入驻，VIP 才收费，99 元/月给 V 标 + 每天自动置顶 1 次）
Day 9-10: 简历置顶（求职端 9.9 元/周，C 端付费破冰）
Day 11-15: 微信 / 支付宝 H5 支付接入
```

**10 天开发，第一个月即可产生现金流**。

### 4.4 商业化分层设计

| 层级 | 对象 | 收费 | 时段 |
|---|---|---|---|
| Banner 广告 | 任意商家 | 50-100 元/周 | +1 月 |
| 付费置顶 | 招聘方 / 房东 / 二手 | 9.9 元/天 | +1 月 |
| 简历置顶 | 求职者 | 9.9 元/周 | +3 月 |
| VIP 会员 | 招聘方 / 商家 | 99 元/月 | +3 月 |
| 商家主页 | 实体商家 | 免费入驻 / VIP 解锁装修 | +3 月 |
| 担保交易抽佣 | 二手买卖 | 1-3% 抽佣 | +12 月 |
| 信息流广告 | 任意 | CPC/CPM | +6 月 |
| 林下经济特产抽佣 | 农户 | 5% 抽佣 | +6 月（季节性） |

### 4.5 商业化的"鸡和蛋"突破

- **破冰点**：第一个付费商家 → 给 1 个月免费 VIP + 承诺首页曝光 → 拿到第一张"付费截图" → 复制到朋友圈 → 第二批商家转化
- **节奏**：先招聘端（最舍得花钱）→ 再商家端（地图/中介/装修）→ 最后二手端（黄牛/批发商）
- **避坑**：不要上来就收"入驻费" → 招不到商家；先免费入驻，VIP 解锁高级功能

## 12. TOP 3 立刻要做

### 🥇 第一件：**IM 即时聊天 + 拨打电话按钮**

- **为什么**：当前用户登录后**联系不到卖家**（手机号脱敏了，没 IM），整个平台的"成交闭环"是断的。再多流量进来也是"看了就走"。
- **做什么**：WebSocket IM + 详情页"联系卖家"按钮（登录后弹出 IM 或显示脱敏手机号 + 拨号按钮）+ 移动端 H5 兼容。
- **投入**：5 天开发 + 1 天测试 = 1 周
- **产出**：**平台从"信息展示"变成"成交撮合"，转化率提升 5x**。
- **依赖**：Phase 1 完成后立即开始。

### 🥈 第二件：**SEO 三件套 + 百度主动推送**

- **为什么**：伊春 30%+ 流量来自百度。**当前平台对百度几乎不可见**——sitemap 0、TDK 0、URL 不友好、结构化数据 0。新站没外链，必须主动推送。
- **做什么**：
  - 25 个分类独立 TDK
  - 12 个区县独立 TDK
  - URL 静态化（`/posts/[type]/[id]-[slug].html`）
  - sitemap.xml + 自动生成 + 百度站长平台提交
  - JSON-LD 结构化数据（JobPosting / Product / RealEstateListing）
  - 面包屑 + 相关推荐
- **投入**：1 周开发 + 1 周收录等待期
- **产出**：**6 个月后百度日 UV 从 0 → 500-1000**（新站冷启动的唯一流量入口）。
- **依赖**：上线第一天就要做，**否则错过新站 3 个月的"沙盒期"红利**。

### 🥉 第三件：**最小可行商业化**（付费置顶 + 商家入驻 + VIP 雏形）

- **为什么**：PRD 把商业化推到 V2 是**最大战略错误**——伊春年广告盘子只有 100-300 万，**纯靠 Banner 养不活团队**。必须前置到 V1.0.5。**没现金流 = 团队散伙 = 项目死掉**。
- **做什么**：
  - 付费置顶（招聘 / 租房 / 便民，9.9 元/天，列表前 3 位 = 付费位）
  - 商家入驻（免费入驻，VIP 才收费——99 元/月给 V 标 + 每天自动置顶 1 次 + 商家主页装修）
  - 简历置顶（求职端 9.9 元/周，C 端付费破冰）
  - 微信 / 支付宝 H5 支付（先用 H5，别等小程序）
  - Banner 广告位（首页 banner 100 元/周，分类页 banner 50 元/周）
- **投入**：2 周开发 + 1 周支付接入
- **产出**：**第一个月营收 1000-3000 元，3 个月后 5000-10000 元，6 个月后 2-3 万元**——是团队活下去的"续命钱"。
- **依赖**：和 IM 同步推进，因为商家付费的最大动力就是"能用 IM 直接聊客户"。
