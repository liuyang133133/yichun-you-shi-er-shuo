## 🚨 双重根因

### 根因 1: 4 个 admin notification 端点无 DTO

`AdminNotificationTemplateController` 有 4 个 raw `@Body() body: { ... }`:

| 端点 | 字段 |
|---|---|
| POST /admin/notifications/templates | event, channel, key, title, body, variables, enabled, priority |
| PATCH /admin/notifications/templates/:id | title, body, variables, enabled, priority |
| POST /admin/notifications/templates/:id/preview | Record<string, string> |
| POST /admin/notifications/broadcast | title, body, event, role, payload, priority |

→ 超长 body / 无效 event 枚举 / priority 越界都会 500。

### 根因 2 (隐藏更严重): AdminNotificationModule **从未注册**!

```bash
$ grep -rn "AdminNotificationModule" backend/src/
backend/src/modules/admin/notification/admin-notification.module.ts:13:export class AdminNotificationModule {}
# ↑ 仅定义, 从未被任何地方 import
```

`admin.module.ts` 漏掉 `AdminNotificationModule` → **所有 6 个 `/admin/notifications/*` 端点全 404**:

```
GET  /admin/notifications/templates       404
POST /admin/notifications/templates       404
GET  /admin/notifications/templates/:id   404
PATCH /admin/notifications/templates/:id  404
POST /admin/notifications/templates/:id/toggle  404
POST /admin/notifications/templates/:id/preview 404
POST /admin/notifications/broadcast       404
```

历史 P0 — admin 通知管理**完全不可用** — 一直没被发现。

## ✅ 修复

### 修复 1: 注册 AdminNotificationModule

`backend/src/modules/admin/admin.module.ts`:

```diff
+ // [V1.1-fix] AdminNotificationModule 之前漏注册, 导致 /admin/notifications/* 全 404
+ import { AdminNotificationModule } from './notification/admin-notification.module';
  ...
  imports: [
    ...
    AdminLoginLogModule, // T-006
+   AdminNotificationModule, // [V1.1-fix] 注册通知模板/广播 controller
    RbacModule,
    AuthModule,
  ],
```

### 修复 2: 新建 `admin-notification-template.dto.ts` (120 行)

4 个 DTO, 字段长度对齐 `NotificationTemplate` (VarChar 200) 和 `Notification` (VarChar 200) schema:

| DTO | 字段 | 限制 |
|---|---|---|
| AdminNotificationTemplateCreateDto | event | @IsIn(8 codes) |
| | channel | @IsIn(site/email/sms/push) |
| | key | @MaxLength(50) |
| | title | @MaxLength(200) |
| | body | @MaxLength(5000) |
| | variables | @IsObject (optional) |
| | enabled | @IsBoolean (optional) |
| | priority | @Min(1) @Max(5) (optional) |
| AdminNotificationTemplateUpdateDto | title/body/variables/enabled/priority | 同上 (optional) |
| AdminNotificationTemplatePreviewDto | Record<string, string> | 透传 |
| AdminNotificationBroadcastDto | title/body/event/role/payload/priority | event @IsIn(8 codes), role @IsIn(user/admin) |

### 修复 3: Controller 应用 DTO

4 个端点 `@Body() body: { ... }` → `@Body() body: AdminNotificationXxxDto`
