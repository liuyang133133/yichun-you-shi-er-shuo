## 关键设计决策

1. **不重写所有组件的 `dark:` 变体** — 利用既有 CSS 变量（globals.css 的 `:root`/`.dark` 两套），所有 token 自动切换。硬编码 `bg-white/15` 的位置是装饰元素（hero 模糊、价格标签背景），是有意为之。
2. **三态循环而非两态** — light/dark/system,符合 OS 习惯
3. **SSR 占位符** — `mounted` flag 避免 hydration mismatch
4. **delete 死配置而非修复** — 没有 source 使用 proxy,留着就是给未来挖坑
